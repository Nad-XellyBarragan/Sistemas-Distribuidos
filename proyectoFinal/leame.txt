Para compilar este proyecto:

1. Estando en la carpeta del proyecto crea el entorno virtual
    Eliminar todo entorno virtual : rm -rf venv
    python3 -m venv venv

2. Activa el entorno virtual
    source venv/bin/activate

3. Dentro del entorno virtual
 - Instala gRPC
    pip install --upgrade pip
    pip install grpcio grpcio-tools

 - Genera los stubs cada que cambies los .proto
    Tracker: python3 -m grpc_tools.protoc -I. --python_out=. --grpc_python_out=. tracker.proto
    Peer: python3 -m grpc_tools.protoc -I. --python_out=. --grpc_python_out=. peer.proto

4. Corre el tracker (siempre dentro del venv)
    python tracker_server.py [IP_TRACKER] [PUERTO]

5. Correr el peer
    python3 peer_node.py [IP_TRACKER] [PUERTO_TRACKER] [PUERTO_PEER]
